# ProLog (Your Career Log from Amateur to Pro)

**AI Career Agent** that helps job seekers (university students) create personal branding content for SNS (Instagram, Blog, LinkedIn) by analyzing their uploaded photos (awards, certificates, activity photos).

## Technical Stack
- React (Vite)
- Tailwind CSS
- Lucide-React
- JavaScript (JSX)

## Deployment
This project is set up for deployment on Cloudflare Pages.
Build output directory should be `dist`.
